# Reversal environment 
Parameters used for generating plots on figure 3

## Task parameters
- Number of observations : 600
- Reversal after 300 obs
- Lambda increase after 200 obs
- Von Mises distributed obs centered at $\frac{\pi}{2}$ and $-\frac{\pi}{2}$ 
- Concentrations parameters = 0.5, 1, and 2 (decreasing width)

## Common parameters
- Concentration of the observer = true concentration value
- Choice variability $\beta = 5.0$
- Initial precision $a_0 = 10$
- Leak $\lambda = 0.05$ initially then $\lambda = 0.1$ during the lesion

## Commitment model parameters ##
- $\tau = 0.2$
- $\theta = 100.0$
